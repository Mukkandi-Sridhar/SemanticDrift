# Security Policy

## Supported Versions

Currently, this repository is for research and documentation purposes. When the implementation and benchmarking codebase is released, supported versions will be listed below.

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover any security vulnerabilities in the proposed frameworks, datasets, or (in the future) code, please report them directly via email.

**Do not report security vulnerabilities through public GitHub issues.**

Please send an email to: [sridharraoyalls@gmail.com](mailto:sridharraoyalls@gmail.com)

We will endeavor to respond to your report within 48 hours and provide updates as the issue is investigated and resolved.

## Responsible Disclosure
We ask that you follow responsible disclosure guidelines and give us time to investigate and patch the vulnerability before disclosing it publicly.
